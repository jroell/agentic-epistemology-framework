# Agentic Epistemology Framework (AEF) - TypeScript Implementation

## Project Structure

```
aef/
├── package.json
├── tsconfig.json
├── jest.config.js
├── README.md
├── src/
│   ├── index.ts                     # Main entry point for the library
│   ├── types/                       # Type definitions
│   │   ├── index.ts                 # Re-exports all types
│   │   ├── common.ts                # Common type definitions
│   │   └── thresholds.ts            # Confidence threshold types
│   ├── core/                        # Core framework components
│   │   ├── index.ts                 # Re-exports all core components
│   │   ├── entity.ts                # Entity interface
│   │   ├── agent.ts                 # Agent implementation
│   │   ├── perception.ts            # Perception classes
│   │   ├── context.ts               # Context implementation
│   │   ├── memory.ts                # Memory interface and implementations
│   │   ├── registry.ts              # Registry implementation
│   │   └── utils.ts                 # Utility functions
│   ├── epistemic/                   # Epistemic components
│   │   ├── index.ts                 # Re-exports all epistemic components
│   │   ├── belief.ts                # Belief implementation
│   │   ├── justification.ts         # Justification and JustificationElement classes
│   │   ├── frame.ts                 # Frame abstract class and implementations
│   │   └── conflict.ts              # Epistemic conflict handling
│   ├── action/                      # Action-related components
│   │   ├── index.ts                 # Re-exports all action components
│   │   ├── action.ts                # Action abstract class
│   │   ├── tool.ts                  # Tool interface and implementations
│   │   ├── capability.ts            # Capability definition
│   │   ├── message.ts               # Message implementation
│   │   ├── goal.ts                  # Goal implementation
│   │   └── plan.ts                  # Plan implementation
│   └── observer/                    # Observer pattern implementation
│       ├── index.ts                 # Re-exports all observer components
│       ├── observer.ts              # Observer interface
│       ├── default-observer.ts      # Default observer implementation
│       └── event-types.ts           # Event type definitions
├── examples/                        # Example use cases
│   ├── simple-agent.ts              # Simple agent example
│   ├── multi-agent-negotiation.ts   # Multi-agent negotiation example
│   └── frame-switching.ts           # Frame switching example
└── tests/                           # Unit and integration tests
    ├── core/                        # Tests for core components
    ├── epistemic/                   # Tests for epistemic components
    ├── action/                      # Tests for action components
    └── integration/                 # Integration tests
```
